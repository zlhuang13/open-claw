## Movie Finder Skill 改进

### 匹配算法修复
- **问题**：search_custom_sites.js 用 `text.includes(queryTerms)` 做子串匹配，对短查询（如"哈利波特7"）太松，导致返回热播推荐而非搜索结果
- **修复**：
  1. 匹配前 normalize 去掉所有标点空格（包括 `·`），两侧 text 和 query 都去掉空格再匹配
  2. 去掉 fallback（不再用 links[0]），没匹配到返回 null
  3. 支持别名数组参数，第一轮用主标题，失败用别名重试，两轮都没有就放弃
- **LLM 预处理**：片名展开由 LLM 负责（"哈利波特7" → "哈利波特 死亡圣器"），脚本只做字符匹配
